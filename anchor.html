Lets learn working of anchor tag and href(Hyper - Reference) Anchor Tag is a tag used to link your page with another page.
It is a container Tag so closing of the tag is required.
To go to another page within the anchor tag linking is required by the help of hyperreference which is denoted by 'href'.
Within href we have to put the nameof our file or page we want to get in.

SO LETS GO TO THE PAGE OF TABLE WITH ADDRESS table.html and we have made our link NEXT within the anchor tag.
