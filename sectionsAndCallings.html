PRIMARY MEMORY : -

  • It is also known as main memory of computer system.
    Primary memory is used to store data and instructions of a program during computer operations.
  • It uses semiconductor technolgy and hence is called as semiconductor memory.

  • Primary memory is of two types :-

    1. RAM,SRAM,DRAM

    2. ROM,PROM,EPROM,EEPROM

    RAM:

    ROM(Random access memory.)

    1. IT is a non volatile memory.
      IT means it stores information even when the power supply is failed.
    2. Rom is used to store information i.e. used to operate system.
    3. As name suggest READ ONLY , here we can only read data stored on it.
    4. Information stored in ROM is in binary format.
      IT is also known as permanent memory.

INTERNAL MEMORY :

  1. CAChe memory is example of internal memory.
  2. CACHE memory is very high speed and special type of memory.
  3. It is used to speed up and sycnhronize with high speed cpu.
  4. It act as buffer b/w RAM and CPU.
  5. It holds frequently requested data, so that immediately available for CPU when asked.
  6. Cache memory is used to reduce average time to access data.
  7. There are various different independent cache in CPU.
  8. The performance of CACHE is measured in HIT RATIO.
  9. If the processor needs to read or write a location it goes to cache.
    If Cache finds the location for processor then CACHE HIT occurs,
    ELSE cache miss
  10. THERE ARE TWO TYPES OF CACHE:-
    • PRIMARY CACHE:- It is located on Processor chip.It has the speed like of registers
    • SECONDARY CACHE:- It is located somewhere b/w primary cache and rest all memory.It is reffered as level2(L2) cache.
